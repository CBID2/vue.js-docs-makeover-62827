## Install Vue.js

If you want to install Vue locally, choose from several methods depending on your needs:

<Aside type="note">
	Be sure to have the latest version of [Node.js](https://nodejs.org/) installed
	on your machine.
</Aside>

<Tabs>
    <TabItem label="npm">
        
        npm create vue@latest
        
    </TabItem>
    <TabItem label="pnpm">
        
        pnpm create vue@latest
        
    </TabItem>
    <TabItem label="yarn">
        
        yarn create vue
        
        <Aside type="note">
            If you are using Yarn and want to use different version of Vue, you can specify the version like this:

            # For Yarn (v1+)
            yarn create vue

            # For Yarn Modern (v2+)
            yarn create vue@latest

            # For Yarn ^v4.11
            yarn dlx create-vue@latest

        </Aside>
    </TabItem>
    <TabItem label="bun">

        bun create vue@latest

    </TabItem>

</Tabs>

Once the setup is complete, do the following commands to install the dependencies and start the development server:

<Tabs>

cd <your-project-name>
<TabItem label="npm">sh npm install & npm run dev </TabItem>

<TabItem label="yarn">sh yarn install & yarn dev </TabItem>
